⚛️ 백준

    🪄 투포인터

	* 배열 합치기
		- a 배열 포인터 i, b배열 포인터 j (a, b는 정렬된 상태)
		- a[i] < b[j] 이면 result에 a[i] 추가, i += 1
		- a[i] >= b[j] 이면 result에 b[j] 추가, j += 1

	* 블로그
		- 누적합

	* 겹치는건 싫어
		- 투포인터
		- 초기값 i = j = 0
		- count[a[j]] < k -> count[a[j]] += 1, j += 1
		- count[a[j]] >= k -> i += 1, count[a[i]] -= 1
