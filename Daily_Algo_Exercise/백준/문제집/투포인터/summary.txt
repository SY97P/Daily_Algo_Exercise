    🪄 투포인터

	* 배열 합치기
		- a 배열 포인터 i, b배열 포인터 j (a, b는 정렬된 상태)
		- a[i] < b[j] 이면 result에 a[i] 추가, i += 1
		- a[i] >= b[j] 이면 result에 b[j] 추가, j += 1

	* 겹치는건 싫어
		- 투포인터
		- 초기값 i = j = 0
		- count[a[j]] < k -> count[a[j]] += 1, j += 1
		- count[a[j]] >= k -> i += 1, count[a[i]] -= 1

	* 두 용액

		- a[i] + a[j] > 0 -> j 감소

	* 회전초밥

		- 0 <= i < n -> j = i + k
		- count[d+1] 리스트

	* 부분합
		- 누적합 asum[n+1]
		- i, j = 0, 1
		- sum[j] - sum[i] < s -> j 증가
		- sum[j] - sum[i] > s -> i 증가
		- sum[j] = sum[i] = s -> result 갱신
