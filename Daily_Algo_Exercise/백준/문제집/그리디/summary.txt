⚛️ 백준

    🪄 그래프순회

    * 거스름돈
        1번 조건. if n >= 10 and (n-10) % 2 == 0 -> 5원 동전 2개 사용
        2번 조건. if n >= 5 and (n-5) % 2 == 0 -> 5원 동전 사용
        3번 조건. if n < 2 or n % 2 != 0 -> break
        4번 조건(default). -> 2원 동전 사용

    * 폴리오미노
        - "AAAA", "BB" 문자열을 이용해서 입력으로 주어진 보드를 채우는 게임
        - "."는 건드리면 안 됨.
