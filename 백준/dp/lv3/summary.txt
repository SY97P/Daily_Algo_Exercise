	🪄 동적계획법3 (비트마스크 DP)

	* 집합
		- 비트마스크 (2진수)를 써야 풀 수 있는 문제 (메모리초과 방지)
		- 기본 자료형 set을 사용하면 못 품.
		- (1 << 5) -> 0b100000 (뒤에 0 5개)
		- (1 << 5) - 1 -> 0b11111 (1 5개)
		- all : (1 << 20) - 1
		- empty : 0
		- add : s | (1 << x - 1)
		- remove : s & ~(1 << x - 1)
		- check : s & (1 << x - 1) == 0 ? 0 : 1
		- toggle : s ^ (1 << x - 1)
